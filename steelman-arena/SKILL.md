# Steelman Arena

Most "pros and cons" lists are theater: weak arguments on the side the person already rejected. This skill forces genuine intellectual combat and — more importantly — finds the **crux**: the underlying factual question that, if answered, would flip the decision.

## Format

### Round 0 — State the motion
Rewrite the user's question as a crisp debatable motion ("This house believes X"). If their question hides two questions, split it and say so.

### Round 1 — Steelman A (the side the user leans AWAY from, first)
Argue it as its smartest, most honest advocate would — not a strawman in a suit. Rules:
- 3 arguments max, strongest first. No filler arguments to pad the count.
- Each argument must include the *mechanism* (why it would be true), not just the claim.
- Include the best available evidence or example; if evidence is weak, admit it inside the argument — real advocates concede weak points to protect strong ones.

### Round 2 — Steelman B
Same rules. Crucially: B may NOT simply negate A's points. It must bring its own positive case.

### Round 3 — Clash
Now pit them directly. For each of A's arguments, give B's best rebuttal, and vice versa — but only where a real rebuttal exists. Mark unanswered arguments explicitly: "B has no good answer to A2." Unanswered arguments decide debates.

### Round 4 — Crux extraction
This is the payoff. Identify 1–3 **cruxes**: statements where
> "If this turned out true, side A wins; if false, side B wins — and reasonable people currently disagree about it."

For each crux, propose the **cheapest real-world test** that would resolve it (a search, an experiment, a conversation, a 1-week trial). A debate that ends without a testable crux has failed.

### Round 5 — Judge's card
- Which side won on the arguments *as presented* (this can differ from what the user should do).
- Confidence: would a small new fact flip it?
- What the user's stated position seems to actually be protecting (a value, a fear, an identity) — name it gently, once, without psychoanalyzing.

## Rules of honor
- Never let the user's visible preference contaminate the steelmen. If anything, over-invest in the side they dislike.
- "It depends" is banned from the judge's card unless followed immediately by *on exactly what*.
- If one side is genuinely indefensible (flat-earth tier), say so in Round 0 and offer the strongest *adjacent* defensible position instead of performing a fake debate.
